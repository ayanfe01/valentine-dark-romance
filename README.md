# Valentine (Dark Romantic Luxury)

## Quick start
1. Install Node.js (18+ recommended)
2. In this folder:
   ```bash
   npm install
   npm run dev
   ```
3. Open http://localhost:3000

## Customize (important)
- Edit the name + letter text: `app/letter/page.tsx`
- Edit the memories/timeline: `app/timeline/page.tsx`
- Optional music: add `public/music.mp3` (then use the Play button on the Letter page)

## Deploy (Vercel)
- Push to GitHub, import to Vercel, deploy.
